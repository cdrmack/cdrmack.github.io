amfora

If you want to play around with the Gemini Protocol on FreeBSD you can use program called amfora to view Gemini Capsules. It's a terminal browser easily available via pkg(8). 

pkg install amfora

When I run amfora I can press ? to see list of shortcuts.

My most used shortcuts:

  • 1-0 go to links 0-10
  • SPC enter address or link ID
  • b go back in history
  • f go forward in history
  • C-b open bookmarks
  • C-d add, change or remove bookmark

If I want to go to the link "11" I press SPC and then provide link ID (11).
